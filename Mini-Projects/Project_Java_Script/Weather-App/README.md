# 🌦️ Weather App باستخدام JavaScript و Fetch API

This project is a simple weather application built with JavaScript that fetches real-time weather data from a weather API and displays it dynamically.

## 🚀 Features

* Search for weather by city name
* Fetch real-time weather data from API
* Display key information such as:

  * Temperature
  * Weather condition
  * City name
* Dynamic UI updates based on user input

## 🛠️ Technologies Used

* JavaScript (Fetch API)
* HTML
* CSS

## 📡 How It Works

* The user enters a city name
* A request is sent to a weather API using `fetch()`
* The response is processed and displayed on the page dynamically

## ▶️ How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Enter a city name to get weather data

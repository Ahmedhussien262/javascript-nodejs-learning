# 📝 To-Do List باستخدام JavaScript

This project is a dynamic To-Do List application built with JavaScript to manage daily tasks with persistent data storage.

## 🚀 Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Save tasks using Local Storage
* Data persists after page reload
* Optional: Cookie-based storage support

## 🛠️ Technologies Used

* JavaScript (ES6+)
* HTML
* CSS
* Local Storage API
* Cookies (for additional storage handling)

## 📡 How It Works

* The user adds a task via input field
* Tasks are stored in **Local Storage**
* On page reload, tasks are automatically loaded
* Users can mark tasks as completed or delete them
* (Optional) Cookies can be used to store small task data

# 🔍 GitHub User Search باستخدام Fetch API

This project is a simple web application built with JavaScript that allows users to search for GitHub profiles using the GitHub API.

## 🚀 Features

* Search for any GitHub user by username
* Fetch real-time data from GitHub API
* Display user information such as:

  * Username
  * Profile picture
  * Public repositories
  * Followers / Following
* Clean and simple UI

## 🛠️ Technologies Used

* JavaScript (Fetch API)
* HTML
* CSS

## 📡 How It Works

* The user enters a GitHub username
* A request is sent to the GitHub API using `fetch()`
* The response is processed and displayed dynamically on the page

## ▶️ How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Enter a GitHub username and search
